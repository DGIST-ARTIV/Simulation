# Simulation
DGIST ARTIV simulation repository

[Carla Simulation](https://carla.org)긴급정지와 같이 실제 차량에 탑제해 테스트시 위험요소가 있는 알고리즘을 시뮬레이션에 먼저 탑제하여 시험하기 위해 개발하기 시작하였다.

현재는 더 나아가 HILs, VILs 시스템을 구성했다.
